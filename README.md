# AlumAssist - Alumni Career Copilot

An intelligent, direct chatbot assisting students to clear roadmap confusion, evaluate skill roadmaps, and acquire internships without motivational fluff.

## 🚀 Features
- **Brutally Honest Direct Career Guidance:** Sharp directions framing placement roadmaps simply.
- **Clean, Minimal Layouts:** Absolute frame-backing contrast popping assistant bubble containers fluidly.
- **Micro Interactivity setup:** Pulsing online badges, smooth loaders ("Yashvi is thinking..."), & visual overlays framing confirmation setups.
- **Action Shortcuts setup:** Fast suggestions chips setup effortlessly support immediate responses loading triggers.

## 🛠️ Tech Stack
- **Backend Frame:** FastAPI (Python)
- **AI Inference Framework:** Groq API (Running Llama-3.3-70b-versatile with rule weights framing prompt boundaries).
- **Frontend Frame:** Vanilla HTML5, Vanilla JavaScript (ES6+), Vanilla CSS3 framework assets.
- **Interactive Libraries Included:** AOS (Animations), Luxon (Time Formatting), Tippy.js (Tooltips), SweetAlert2 (Modals support buffers).

## ⚙️ Installation & Usage

1. **Clone the Repository:**
   ```bash
   git clone <repository_url>
   cd AlumAssist
   ```

2. **Setup Environment:**
   Create a `.env` file in the root containing your Groq Inference API Key:
   ```env
   GROQ_API_KEY=your_actual_groq_api_key_here
   ```

3. **Install Dependencies:**
   Run the following (Ideally inside a virtual environment setup framework):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

4. **Run Server Instance Container:**
   ```bash
   python main.py
   ```
   The application begins serving locally at `http://127.0.0.1:8000/` setups accurately.
